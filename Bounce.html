Bounce Synopsis

Made in 2 weeks as a self development challenge, Bounce is my second mobile game that tests my knowledge of game development against the different platform design choices. I made a small minigame type game where the player must bouce from side to side to gain points while obstacles fall from the sky. With features like leveling and skins to give extra reasons to play.

Some features this game has:

  • Small but replayable gameplay loop
  • Levelling system for progress
  • 25+ Skins to unlock
