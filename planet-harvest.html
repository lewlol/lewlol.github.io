Planet Harvest Synopsis

Grade: DISTINCTION

Made in 2 Months for a college project, Planet Harvest is my first mobile game. Which I believe was successful. The module asked me to explore a new skill and document this through social media. I was tasked to use the power of social media to push a project. I decided that I would make this game for mobile as my new skill using Development Blogs to show my progress.

Some features this game has:

  • Many planets to explore through
  • Surplus of upgrades
  • Upgrade shop to increase production on planets
  • Rare artefacts and crystals to collect along the way
  • Mobile game features like daily login and offline production.
