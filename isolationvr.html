IsolationVR Synopsis

IsolationVR was my first attempt at developing in Virtual Reality prototype. I set the task of creating a wave shooter in VR and attempting to create systems like guns combines with VR controls and aiming. I learnt some really interesting development changes in Virtual Reality like how the controls work and movement systems. I feel this prototype shows my progress in learning how to develop in VR.
